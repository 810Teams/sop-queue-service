# Comment Service

### POST /Comment

ส่ง comment ขึ้นไปบน service

```json
{
        "customer": "String", // customer name
        "name": "String", // shop name
        "review": "String"
}
```

### GET /Comment

ทำการดึง Comment จาก Service 

### GET /Comment/ชื่อร้าน

ทำการดึง Comment จากชื่อร้าน บน service
