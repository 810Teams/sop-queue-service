# API Endpoints : Inventory Service

## Shop-service

- `POST   /shops`
- `GET    /shops`
- `GET    /shops/:shop_id`
- `PATCH  /shops/:shop_id`
- `DELETE /shops/:shop_id`

## Product-service

- `POST   /products`
- `GET    /products`
- `GET    /products/:shop_id`
- `PATCH  /shops/:shop_id`
- `DELETE /shops/:shop_id`
